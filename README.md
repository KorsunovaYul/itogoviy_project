# МатемаТест - Образовательная платформа с тестами

## Описание проекта

МатемаТест — это образовательная веб-платформа для прохождения тестов по математике. Платформа позволяет студентам проверить свои знания в различных разделах математики с ограниченным временем на выполнение и автоматическим подсчетом результатов.

## Функциональность

- **Система тестирования** - три готовых теста по различным темам математики
- **Таймер обратного отсчета** - ограниченное время на каждый тест (15 минут)
- **Прогресс-бар** - визуальное отображение прогресса прохождения теста
- **Навигация по вопросам** - возможность переключаться между вопросами
- **Подсчет баллов** - автоматический подсчет правильных ответов
- **Детальные результаты** - анализ каждого ответа с правильными решениями
- **Адаптивный дизайн** - работает на всех устройствах

## Темы тестов

1. **Алгебра** - уравнения, неравенства, системы
2. **Геометрия** - площади, объемы, теоремы
3. **Тригонометрия** - тригонометрические функции и формулы

## Технологии

- **HTML5** - разметка страницы
- **CSS3** - стилизация и адаптивный дизайн
- **JavaScript (ES6+)** - логика приложения

## Как добавить новый тест

В файле `data.js` добавьте новый объект теста:
\
```javascript
{
  id: 4,
  title: "Название теста",
  description: "Описание теста",
  duration: 15,
  questions: [
    {
      id: 1,
      question: "Текст вопроса",
      options: ["Вариант 1", "Вариант 2", "Вариант 3", "Вариант 4"],
      correct: 0 // индекс правильного ответа
    }
  ]
}
```
